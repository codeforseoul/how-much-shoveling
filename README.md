# What is this?

이 저장소는 코드어크로스 행상에서 만들어진 아이디어를 구현하기 위한 저장소입니다.
가칭 How Much Shoveling We've Done.

이 프로젝트는 정부의 재정데이터중 건설관련 비용을 지도상에 표시하여 보다 쉽게 건설관련 비용의 흐름과 유용성을 보여주는 서비스입니다.
